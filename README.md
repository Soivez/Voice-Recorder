# Voice-Recorder
Приложение для ведение голосовых записей. Для хранения файлов использую mp3 формат (только в конце узнал, что vk.com/docs не поддерживает его, но можно поменять на что-то другое). Файлы хранятся в internal storage + room для вспомогательной информации и хранения названий файлов. Есть поддержка авторизации через VK_SDK, так же получение данных из документов вк.
